# React CDN application
Application using react, react dom and babel CDN to filter data.

## Dependencies

- react
- react-dom
- babel

## How to run the app

Option 1:  

Run the index.html using **Live server** extension in VS Code.  

Option 2:  

Deploy the application on a web server
